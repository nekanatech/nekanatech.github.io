Cooking up the cocaine

Okay first we’re gonna need to actually make/cook the cybercocaine. Here are the ingredients (btw, I’m doing this on a Lunix system, so results might vary if you’re using Windoze):

  • mcs (or anything that’ll compile C#)
  • some pic that looks like it was made by stoners (I’ll use this one)
  • An icon of a cake (just Google search “cake” and convert it into an icon with Photoshop/GIMP/online tools).
  • A USB thumb drive

$ mcs -out:cybercocaine.exe /reference:System.Windows.Forms.dll /reference:System.Drawing cybercocaine.cs

Alright, now if you managed to compile it and you got your stoner pic, let’s load up the syringe!

Loading up the syringe

We’re gonna need a USB thumb drive for this. Reformat it to NTFS and then get ready to load it up with cybercocaine by copying the cybercocaine.exe file, the stoner image file, the cake icon and by creating an autorun file.

Keep a few things in mind about the file naming scheme:

  • cybercocaine.exe can be called whatever you want. I renamed mine to cake.exe
  • The stoners image MUST be called image.jpg
  • The cake.ico file can be named whatever you want.
  • The autorun file MUST be named autorun.inf

So lets create our autorun.inf file:

$ touch autorun.inf

Add this to the autorun file:

[autorun]
Icon=cake.ico
Label=Delicious Cake
;Open=cake.exe
ShellExecute=cake.exe
UseAutoPlay=1

When saving it, make sure that the line formatting is Windoze.
